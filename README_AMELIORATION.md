## 🎓 Types d'Analyses Supportées

Le système détecte automatiquement et adapte la présentation :

### 1. 💸 Non-Recouvrement
**Exemple:** "déclarations non recouvrées d'août 2024"
- Calcul des pertes potentielles
- Taux de recouvrement
- Top taxes à prioriser
- Recommandations de recouvrement

### 2. 💰 Analyse Financière
**Exemple:** "comparaison déclaré vs recouvré"
- Performance par taxe
- Écarts financiers
- Analyses comparatives

### 3. 🗺️ Analyse Régionale
**Exemple:** "recettes du bureau de Dakar"
- Classement des zones
- Top/Bottom performers
- Répartition géographique

### 4. 📅 Analyse Temporelle
**Exemple:** "évolution 2024"
- Tendances temporelles
- Comparaisons périodiques
